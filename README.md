### インストール

1. リポジトリをクローンします
```bash
git clone https://github.com/Yuime-Inc/hr-system-frontend.git
```

2. 依存関係をインストールします
```bash
npm install
```


3. 開発サーバーを起動します
```bash
npm run dev
```

## 使用方法

### 求職者向け
- プロフィール作成と履歴書アップロード
- 求人検索とフィルタリング
- 求人への応募
- 応募状況の追跡

### 企業向け
- 求人投稿
- 応募管理
- 候補者プロフィール閲覧
- 企業ダッシュボード

### 管理者向け
- ユーザー管理
- コンテンツモデレーション
- システム分析
- プラットフォーム設定
